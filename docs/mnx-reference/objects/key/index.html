The key signature object

Type: Dictionary

The key object defines a key signature applicable to this and all following measures, until changed..

Attributes

Name Type Required? Description
color color No An optional color to be used when rendering this key signature.
fifths fifths Yes The number of fifths distance from a signature with no accidentals.

Plus globally available attributes: _c, _x, id

Parent objects

This object is used by the following parent objects:

  • measure global: "key"

Examples

This object is used in the following examples:

Accidentals, Key signatures, Organ layout
