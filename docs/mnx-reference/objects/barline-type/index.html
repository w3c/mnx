The barline type object

Type: String

Allowed values:

Value Description
"dashed"
"dotted"
"double" Two light lines together, conventionally known as a "double barline."
"final" A light line followed by a heavy line, as conventionally used at the end of a score.
"heavy"
"heavyHeavy"
"heavyLight"
"noBarline"
"regular"
"short"
"tick"

Parent objects

This object is used by the following parent objects:

  • barline: "type"

This object is used in the following examples:
