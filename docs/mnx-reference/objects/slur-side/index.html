The slur side object

Type: String

This indicates the orientation of a slur or tie. It specifies whether the slur/tie curves upward or downward.

Allowed values:

Value Description
"down" The slur/tie is curved downward (the tips face up).
"up" The slur/tie is curved upward (the tips face down).

Parent objects

This object is used by the following parent objects:

  • slur: "side"
  • slur: "sideEnd"
  • tie: "side"

Examples

This object is used in the following examples:

Slurs, Slurs (for chords), Slurs (targeting specific notes), Tie targets
