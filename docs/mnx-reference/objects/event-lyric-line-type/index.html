The event lyric line type object

Type: String

Allowed values:

Value Description
"end" This syllable continues a previous syllable and doesn't continue forward to a subsequent syllable.
"middle" This syllable continues a previous syllable and continues forward to a subsequent syllable.
"start" This syllable begins a word and continues forward to a subsequent syllable.
"whole" This syllable constitutes an entire word.
