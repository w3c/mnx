The written object

Type: Dictionary

Information about how this note is written in context of a transposed score or part.

Keys

Name Type Required? Description
_c string No An optional comment. This serves a similar function as XML or HTML comments.
diatonicDelta integer (signed) No After computing the transposed pitch by applying the part's transposition and any necessary "keyFifthsFlipAt" value, this specifies the number of diatonic steps to add or remove to obtain the final written pitch.

If not specified, the value is 0.

For example, in case of a C-flat note that you want to write for the player as a B natural, you'd use "diatonicDelta": -1.
id ID No
