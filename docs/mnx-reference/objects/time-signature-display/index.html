The time signature display object

Type: String

This encodes visual glyphs for a time signature.

Allowed values:

Value Description
"common" The symbol representing "common time."
"cut" The symbol representing "cut time."

Parent objects

This object is used by the following parent objects:

  • time signature: "display"
