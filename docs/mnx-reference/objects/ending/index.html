The ending object

Type: Dictionary

The ending object encodes an alternate ending, sometimes known as a "volta bracket."

Keys:

Name Type Required? Description
"class" style class object No An optional style class to use for this ending.
"color" color object No An optional color to use when rendering this ending.
"duration" ending duration object Yes
"numbers" An array of ending number objects No
"open" ending open object No

Examples

This object is used in the following examples:

Repeats (with alternate endings, advanced), Repeats (with alternate endings, simple)
