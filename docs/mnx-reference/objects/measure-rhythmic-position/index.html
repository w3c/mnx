The measure rhythmic position object

Type: Dictionary

Describes the rhythmic position of a notational object within a piece of music. This is comprised of a measure index plus a rhythmic position within that measure.

Attributes

Name Type Required? Description
measure measure number Yes
position rhythmic position Yes The position within the measure.

Plus globally available attributes: _c, id

Parent objects

This object is used by the following parent objects:

  • layout change: "location"
  • ottava: "end"

Examples

This object is used in the following examples:

Multiple layouts, Organ layout, Ottavas (8va)
