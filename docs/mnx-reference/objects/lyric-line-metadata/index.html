The lyric line metadata object

Type: Dictionary

Keys:

Name Type Required? Description
"label" lyric line label object No A label for this lyric line. This could be used, for example, in software UI to allow for showing/hiding specific lyric lines.
"lang" language code object No A language code that applies to all the lyrics in this lyric line.
