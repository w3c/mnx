The kit component object

Type: Dictionary

Attributes

Name Type Required? Description
name string No A human-readable name for this kit component, e.g., for purposes of display in a UI.
sound ID No ID of the sound used for this kit component. See the "sounds" attribute in "global".
staffPosition staff position Yes

Plus globally available attributes: _c, _x, id
