The tie target type object

Type: String

Allowed values:

Value Description
"arpeggio" The tie is part of an arpeggio notated as consecutive ties, like this:



Note: In this example, the final tie would be encoded as "nextNote".
"crossJump" The tie ends on a note whose event does not directly follow the start note's event, such as an alternate ending, repeat or jump (e.g., D.S. al Coda).
"crossVoice" The tie ends on a note in a different sequence.
"nextNote" The tie ends on a note in the event that directly follows the start note's event.

This is by far the most common type of tie.
