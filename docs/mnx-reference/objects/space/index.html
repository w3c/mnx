The space object

Type: Dictionary

The space object represents a discrete period of time within a sequence in which no sequence content occurs. It embodies the idea of blank space within a measure.

Keys:

Name Type Required? Description
"duration" note value quantity object Yes The metrical duration of this space.

Note that, in contrast to the event's "value", this duration attribute is not constrained to a single note value, but may be a multiple of one.
"type" The string "space" Yes
