The rest object

Type: Dictionary

The rest object defines a rest within an event, along with other information pertaining to the rest rather than to its containing event.

Keys:

Name Type Required? Description
"pitch" pitch object No If provided, this mandates that the rest be placed on the staff line corresponding to the provided pitch. The accidental component of the pitch is ignored for this purpose.

Examples

This object is used in the following examples:

Beams, Beams (across barlines), Beams (secondary beam breaks), Beams (with inner grace notes), Multimeasure rests, Organ layout, Parts, Three-note chord and half rest
