The lyric line object

Type: Dictionary

This encodes a single lyric syllable within a single line (i.e., a single verse).

Keys:

Name Type Required? Description
"hyphen" boolean object No Specifies that this syllable is part of a multi-syllable word and continues into a subsequent syllable. Generally this means software will put a hyphen after this syllable, if space is available.

The hyphen character itself should not be encoded within "text".

Note: For a syllable with "hyphen":true, the subsequent syllable does not define "hyphen": true unless it also continues into yet another subsequent syllable. That is, the "hyphen" relationship is only encoded on the starting syllable.
"text" string object Yes The text of the syllable.

This might be an empty string in the case of lyrics that are skipped for a given verse.
