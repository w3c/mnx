The page object

Type: Dictionary

The page object represents a single page of music â€” either printed (as on paper) or digitally rendered.

It's part of a score and contains a system object for each system of music in the page.

Attributes

Name Type Required? Description
layout ID No An optional layout to use for the systems in this page. If provided, this should be the ID of a system-layout object in the same MNX document.

This overrides the layout specified by the parent score object. It can be further overridden, on a per-system basis, in child system objects.
systems An array of system objects Yes

Plus globally available attributes: _c, id

Parent objects

This object is used by the following parent objects:

  • score: "pages"

Examples

This object is used in the following examples:

Multimeasure rests, Multiple layouts, Orchestral layout, Organ layout, System layouts
