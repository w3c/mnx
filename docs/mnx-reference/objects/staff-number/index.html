The staff number object

Type: Number

This number identifies a staff in context of a multi-staff part. Counting starts at 1 for the top staff.

For example, consider piano music with a grand staff: two staves total. Staff number 1 refers to the top staff, and staff number 2 refers to the bottom one.

Parent objects

This object is used by the following parent objects:
