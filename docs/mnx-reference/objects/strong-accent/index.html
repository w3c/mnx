The strong accent object

Type: Dictionary

Attributes

Name Type Required? Description
_c string No An optional comment. This serves a similar function as XML or HTML comments.
id ID No
pointing up or down No Specifies whether this accent should point upward or downward. If not provided, consuming applications should use their own automatic algorithms to determine this.
