The ottava amount object

Type: Number

Allowed values:

Value Description
-1 The written pitch is one octave higher than sounded. Traditionally rendered with an "8vb" symbol.
-2 The written pitch is two octaves higher than sounded. Traditionally rendered with a "15mb" symbol.
1 The written pitch is one octave lower than sounded. Traditionally rendered with an "8va" symbol.
2 The written pitch is two octaves lower than sounded. Traditionally rendered with a "15ma" symbol.

This object is used by the following parent objects:
