The fine object

Type: Dictionary

Attributes

Name Type Required? Description
class style class No An optional style class to use for this fine direction.
color color No An optional color to be used when rendering this fine direction.
location rhythmic position Yes The location of this fine direction within the bar.

Plus globally available attributes: _c, _x, id

Parent objects

This object is used by the following parent objects:

  • measure global: "fine"
