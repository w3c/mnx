The accidental display object

Type: Dictionary

Keys:

Name Type Required? Description
"cautionary" boolean object No Whether the accidental is considered cautionary (traditionally rendered in parentheses). Default is false.
"editorial" boolean object No Whether the accidental is considered an editorial marking (traditionally rendered in square brackets). Default is false.
"show" boolean object Yes Whether the accidental is to be displayed.
