The dynamic object

Type: Dictionary

Describes a textual dynamic direction to the performer.

Keys

Name Type Required? Description
glyph smufl glyph No A specific SMuFL glyph name to be used when rendering this dynamic.
position rhythmic position Yes The position of this dynamic within the measure.
staff staff number No The staff index to which this dynamic applies. By default, a dynamic applies to all staffs in the part.
value dynamic type Yes
voice voice name No The voice to which this dynamic applies. By default, a dynamic applies to all voices in the part.
