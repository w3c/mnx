The dynamic object

Type: Dictionary

Describes a textual dynamic direction to the performer.

The metrical position of this dynamic direction is determined by sequencing the content.

Keys:

Name Type Required? Description
"glyph" smufl glyph object No A specific SMuFL glyph name to be used when rendering this dynamic.
"type" The string "dynamic" Yes
"value" dynamic type object Yes
