The lyrics object

Type: Dictionary

This contains the lyric data for a given event. Typically this correponds to a single sung syllable.

Keys:

Name Type Required? Description
"lines" An array of lyric line objects No An ordered array of lyric lines (e.g., for multiple verses). The first element in the array corresponds to the top-most line, and so forth.
