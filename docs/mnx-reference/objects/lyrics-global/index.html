The lyrics-global object

Type: Dictionary

Keys

Name Type Required? Description
"lineMetadata" An object with user-defined keys, where each value is a lyric line metadata object No An object mapping lyric line IDs (which are user-defined) to lyric metadata objects. This encodes global metadata on a per-lyric-line basis.
"lineOrder" An array of lyric line ID objects No An array of all lyric line IDs used in this MNX document, in the order in which they should be visually displayed (from top to bottom).
