Octave shift type data type

Base type: Integer (signed)

Allowed values

-8 8va (notes are rendered down one octave from sounding pitch)
8 8vb (notes are rendered up one octave from sounding pitch)
-15 15ma (notes are rendered down two octaves from sounding pitch)
15 15mb (notes are rendered up two octaves from sounding pitch)
-22 22ma (notes are rendered down three octaves from sounding pitch)
22 22mb (notes are rendered up three octaves from sounding pitch)
