MNX specification > MNX reference > Data types > Voice stem direction

Voice stem direction data type

Allowed values

part Use stem directions as defined by the first <part-layout> within this <voice-layout>.
up Set all stem directions to up.
down Set all stem directions to down.
float Determine stem directions algorithmically according to notehead staff position.
