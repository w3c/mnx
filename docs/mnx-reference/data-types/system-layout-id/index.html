System layout ID data type

A string used to identify a system layout in the MNX document. This must contain no spaces, and it is case-sensitive.

Examples: "fullscorelayout", "flutepartlayout".

Minimum length: 1

Where is this type used?

<page> — "layout" attribute

<score> — "layout" attribute

<system> — "layout" attribute

<system-layout> — "id" attribute

<system-layout-change> — "layout" attribute
