Perform option data type

This specifies whether an element is performed by consuming software.

For example, if consuming software offers synthetic playback, specifying perform="none" for a <note> element means the note will be excluded from playback (but still visually rendered).

Allowed values

normal Default. The element is performed.
none The element is omitted from performance.
