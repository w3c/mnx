The <ending> element

Parent element: <directions-global>

The <ending> element encodes an alternate ending, sometimes known as a "volta bracket."

Each alternate ending — say, the first ending to a repeated section — is encoded with two <ending> elements, one with type="start" and another with type="stop".

Content

Always empty.

Attributes

Name Type Required? Description
type Ending type Yes
color Simple color No An optional color to be used when rendering this element and all child elements.
number Ending number No
style Style class No An optional style to use for this element and all child elements.
