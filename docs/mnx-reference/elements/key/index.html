The <key> element

Parent elements: <directions-global>, <directions-part>

The <key> element defines a key signature applicable to this and all following measure content, until changed. This is a staff direction.

The measure location of a <key> direction is ignored and is assumed to be zero.

The staff index of a <key> direction is ignored, as the direction always applies to all the staves in a given part (if not the whole score).

It is invalid to place more than one <key> element within a single <measure> (or within a single <measure-global>).

<key> elements in <global> are required for every key change. <key> elements below <part> are optional; if present, their "fifths" value must either be identical to the corresponding value in the global key or different by a multiple of 12.

Content

Always empty.

Attributes

Name Type Required? Description
fifths Integer (signed) Yes The number of fifths distance from a signature with no accidentals.
class Style class No An optional style class to use for this element and all child elements.
color Simple color No An optional color to be used when rendering this element and all child elements.
