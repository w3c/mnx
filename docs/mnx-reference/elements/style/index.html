The <style> element

Parent elements: <global>, <page>, <score>, <system-layout>, <system-layout-change>

The <style> element is a way to define visual styling within an MNX document. Its required selector attribute specifies which MNX element(s) the style applies to.

See Styling an MNX document for more.

This example defines a style class "emphasized", which means to use the color red:

<style selector=".emphasized" color="#ff0000">

A <note> element can then use the "class" attribute to use this style class:

<note class="emphasized" ...>

This example says "all <note> elements must be blue":

<style selector="note" color="#0000ff">

At the moment, the only available style property is "color".

Content

Always empty.

Attributes

Name Type Required? Description
selector CSS selector Yes A CSS-style selector that specifies which MNX element(s) this style applies to.

Currently MNX supports two types of selectors, element selectors and style selectors.
color Simple color No The color to apply to any elements targeted by this style.
