The <style> element

Parent elements: <global>, <page>, <score>, <system-layout>, <system-layout-change>

The <style> element is a way to define visual styling within an MNX document. There are two types of <style> elements:

* <style class="..."> defines a style class that can be referenced from another MNX element.
* <style element="..."> targets all elements of a given name.

This example defines a style class "emphasized", which means to use the color red:

<style class="emphasized" color="#ff0000">

A <note> element can then use the "style" attribute to use this style:

<note style="emphasized" ...>

This example says "all <note> elements must be blue":

<style element="note" color="#0000ff">

At the moment, the only available style property is "color".

Content

Always empty.

Attributes

Name Type Required? Description
class Style class No The name for this <style> rule. This name can then be used in any styleable MNX element, via the "style" attribute.

This is required if "element" is not provided.
color Simple color No The color to apply to any elements targeted by this style.
element MNX element name No The MNX element that this style rule applies to.

This is required if "class" is not provided.
