The <system-layout> element

Parent element: <layouts>

The <system-layout> element defines a visual layout that can be used by <system> or <score> elements.

For example, a layout might say "Include the vocal part, with the piano part directly below. For the piano part, use a grand-staff brace."

To encode this information, <system-layout> has the child elements <staff-layout> and <group-layout>.

Content

In any order
  • In any order
    • <staff-layout> (Zero or more times)
    • <group-layout> (Zero or more times)
  • <style> (Zero or more times)

Attributes

Name Type Required? Description
id System layout ID Yes A unique identifier for this system layout. This is used by the <system> (or <score>) element to specify which layout the system (or score) uses.
