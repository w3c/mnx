The <page> element

Parent element: <score>

The <page> element represents a single page of music â€” either printed (as on paper) or digitally rendered.

It's part of a <score> and contains a <system> element for each system of music in the page.

Content

<system> (One or more times)

Attributes

Name Type Required? Description
layout System layout ID No An optional layout to use for the systems in this page. If provided, this should be the ID of a <system-layout> element in the same MNX document.

This overrides the layout specified by the parent <score> element. It can be overridden on a per-system basis in child &tl;system> elements.
