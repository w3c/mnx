The <system> element

Parent element: <page>

The <system> element describes a system of notation â€” that is, a collection of staves that represent one or more measures across one or more parts.

This element is used in context of a <page>. Each <system> specifies its starting measure number and can optionally specify a system layout.

Content

<system-layout-change> (Zero or more times)

Attributes

Name Type Required? Description
measure Measure number Yes Specifies the starting measure in this system. This corresponds to the "index" attribute of the <measure> element.
layout System layout ID No An optional layout to use for this system. If provided, this should be the ID of a <system-layout> element in the same MNX document.

This overrides any layout specified by the parent <page> or <score> elements.
