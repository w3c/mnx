The <segno> element

Parent element: <directions> (global)

Content

Always empty.

Attributes

Name Type Required? Description
location Measure location Yes The measure location of this direction.
glyph SMuFL glyph name No A specific SMuFL glyph name to be used when rendering this symbol.
