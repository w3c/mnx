The <system-layout-change> element

Parent element: <system>

The <system-layout-change> element lets you indicate a change in <system-layout> in the middle of a system — usually, but not necessarily, due to changes in stemming.

Content

Always empty.

Attributes

Name Type Required? Description
layout System layout ID Yes
location Measure location Yes
