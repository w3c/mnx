The <time> element

Parent element: <directions> (global)

Content

Always empty.

Attributes

Name Type Required? Description
signature Time signature Yes
measure Time signature No This overrides the notated time signature for the current measure only, where this value differs from "signature". This is of particular use for anacruses and for shortened measures prior to a repeat or jump back to an anacrusis.

The "measure" attribute must be provided in all cases where the actual content of a measure is of a different length from that indicated by "signature". MNX does not require consumer implementations to examine the contents of measures to determine their intended length.

Examples

This element is used in the following examples:

Accidentals, Beams, Beams (across barlines), Beams (hooks), Beams (secondary beam breaks), Beams (with inner grace notes), Dotted notes (augmentation dots), Jumps (D.S. al Fine), Jumps (Dal Segno), Key signatures, Multiple voices, Octave shifts (8va), Parts, Repeats, Repeats (more than once repeated), Repeats (with alternate endings, advanced), Repeats (with alternate endings, simple), Repeats (with implied start repeat), Slurs, Slurs (for chords), Slurs (incomplete slurs), Slurs (targeting specific notes), Three-note chord and half rest, Ties, Time signatures, Tuplets, Two-bar C major scale, “Hello world”
