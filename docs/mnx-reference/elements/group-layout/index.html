The <group-layout> element

Parent elements: <group-layout>, <system-layout>

The <group-layout> element is a way to group <staff-layout> elements together, for the purposes of giving them a unified name and/or grouping symbol.

You can also nest <group-layout> elements within another, to represent more complex groupings of staves, as in large orchestral scores.

Content

In any order

  • <staff-layout> (Zero or more times)
  • <group-layout> (Zero or more times)

Attributes

Name Type Required? Description
label String No An optional name for this group of staves. If provided, this should be rendered to the left of the staff group.
symbol Grouping symbol No The symbol to use along the left vertical edge of this group of staves. If omitted, the default value is "none".

Note that the child <staff-layout> or <group-layout> elements might also define their own own "symbol" attributes.
